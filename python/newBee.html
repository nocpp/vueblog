    # python版本

    • python2.7 老的版本，与新的语法不同，比如print "aaa"
    • python3 新的，print("aaa")

    # 环境

    • 两个版本都安装，命令行名称使用python2和python3区分
    • pip是包管理工具，同样也需要pip2和pyp3。全局安装的

    # pip常用命令

    • pip list // 安装的包有哪些
    • pip show sseclient //显示sseclient安装位置

    • python3 -m site --user-site 查看安装位置
