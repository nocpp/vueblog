    10/31/2021 数组

    # 数组中不会改变自己的方法

    1. filter --> 返回新数组
    2. slice --> 返回数组片断
    3. concat --> 返回连接的数组
    4. join --> 返回数组转为字符串

    # 数组中注意的方法

    1. push方法可以传入多个参数，一次传入多个
    2. sort方法，传入的参数，(a, b) 返回小于0的数，a在前，大于0的数，b在前，等于0，不变
